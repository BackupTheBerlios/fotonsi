  • BaseFormulario: Es la base de toda la jerarquía de objetos. Aquí se definen los atributos y métodos comunes a todas las instancias de los objetos generados.
      setNull / setNoNull: El valor del objeto no puede o puede estar vacío.
      setFocus / setNoFocus: En caso de error, el campo debe coger el foco o no
      getNull / getFocus:: Obtener si el objeto es requerido o no / obtener si el foco se coje antes un error o no.
      AddEvent: Esta es la base para el manejo de los eventos de 'forma transparnte'. La idea en un principio, es que los distintos objetos gestion los distintos eventos.
